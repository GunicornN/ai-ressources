# AI Ressources

Catalogue de ressources IA : AGENTS.md, system prompts, skills, serveurs MCP et guides.

Chaque fichier contient le contenu complet de la ressource avec un frontmatter YAML pour les metadonnees.

## Structure

```
agents-md/       # Specifications AGENTS.md
system-prompts/  # System prompts (Claude Code, Cursor, Devin, etc.)
skills/          # Skills, hooks, slash-commands
mcp-servers/     # Serveurs MCP (officiels et communautaires)
guides/          # Guides et tutoriels
```

## Format des fichiers

```yaml
---
title: "Nom de la ressource"
url: "https://github.com/..."
description: "Description courte"
sourceRepo: "owner/repo"
tags: ["tag1", "tag2"]
date: 2025-01-01
stars: 1234
forks: 56
summary: "Resume IA en francais"
---

Contenu complet de la ressource...
```
